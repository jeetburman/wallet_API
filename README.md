# wallet_API
A simple backend API built with Node.js, Express, and MongoDB that includes:

JWT-based authentication

User signup & signin

Protected routes with middleware

Automatic MongoDB connection and dummy data insertion for testing

Features : ----

User Authentication — Signup & Login with hashed passwords.

JWT Middleware — Protect routes and extract the authenticated user.

MongoDB Integration — Automatically creates database and collection if not present.

Dummy Data — Sample entries inserted for easy visual testing in MongoDB Compass.

Scalable Structure — Can be expanded for new features.
